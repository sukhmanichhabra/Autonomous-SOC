## 🚀 How to Use

### 1. Install PostgreSQL

**macOS:**
```bash
brew install postgresql pgvector
brew services start postgresql
```

**Linux:**
```bash
sudo apt install postgresql postgresql-contrib postgresql-14-pgvector
sudo systemctl start postgresql
```

### 2. Initialize Database

```bash
python my-ai-soc-agent/init_db.py
```

The script will:
- ✅ Create database and user
- ✅ Enable pgvector extension
- ✅ Create LangGraph and RAG tables
- ✅ Update `.env` with `DATABASE_URL`

### 3. Run Your SOC

```bash
# Streamlit dashboard
streamlit run app.py

# CLI
cd my-ai-soc-agent
python main.py --target 192.168.1.1

# Seed threat intelligence
python main.py --seed-db
```

## 🔑 Core Changes

### Before (SQLite + ChromaDB)
```python
from langgraph.checkpoint.sqlite import SqliteSaver
from vector_db.threat_intel_store import ThreatIntelStore

conn = sqlite3.connect("./checkpoints.sqlite")
checkpointer = SqliteSaver(conn)
store = ThreatIntelStore()
```

### After (PostgreSQL + pgvector)
```python
from langgraph.checkpoint.postgres import PostgresSaver
from vector_db.pgvector_store import ThreatIntelStore
from checkpointer import create_postgres_checkpointer

checkpointer = create_postgres_checkpointer()
store = ThreatIntelStore()
```
